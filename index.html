Как работает приложение IStream

Приложение IStream взаимодействует с наземной панелью через TCP, чтобы получать поток данных. Модуль Listener выполняет подключение и принимает данные:

416447
419957
424387
429769
435368
440677
445704
450266
454673
458853

Данные обрабатываются через демодулятор, который реализован модулем Demoduation. Этот процесс осуществляется через метод:

string Demodulate(int signal)

Как только демодулятор обнаруживает фрейм, он возвращает строку:

1100100101111110010001000000001001111110110000000101011111100111010010

Эта строка передается в следующий модуль – Decoder. Его метод:

Frame Decode(string binaryFrame)
